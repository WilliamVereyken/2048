# 2048

A classic 2048 puzzle game implementation.

## About

2048 is a sliding tile puzzle game where the goal is to combine numbered tiles to reach the 2048 tile.

## How to Play

- Use arrow keys (or swipe on touch devices) to move tiles
- When two tiles with the same number touch, they merge into one
- Try to create a tile with the number 2048 to win!

## Getting Started

To play the game locally:

1. Open `index.html` in your web browser
   - You can double-click the file, or
   - Right-click and select "Open with" → your preferred browser
   - Or drag and drop the file into your browser window

2. Use your arrow keys to play:
   - ↑ Up arrow - Move tiles up
   - ↓ Down arrow - Move tiles down
   - ← Left arrow - Move tiles left
   - → Right arrow - Move tiles right

3. The game automatically saves your best score in your browser's local storage

## Features

- Classic 2048 gameplay
- Arrow key controls (perfect for Mac)
- Touch/swipe support for mobile devices
- Score tracking with best score persistence
- Win and game over detection
- Smooth animations
- Responsive design
